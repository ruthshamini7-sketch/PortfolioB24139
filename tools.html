Technologies Used in the Project

To build this digital portfolio, I used a combination of *frontend and backend technologies, along with tools to design and enhance user experience. The goal was to make the portfolio **interactive, visually appealing, and functional*.

  • HTML (Hypertext Markup Language): Used to create the structure of the website. Example: All pages like index.html, problem.html are written in HTML.
  • CSS (Cascading Style Sheets): Used for styling and layout. Example: I used CSS to make dark backgrounds, color themes, fonts, buttons, and hover effects for each page.
  • JavaScript: Used to add interactivity and dynamic features. Example: Console messages, alert messages, and navigation animations are handled using JS.
  • Notepad: Simple text editor for writing HTML, CSS, and JS code.
  • Images & Videos: Used for visual representation. Example: Screenshots of project results and a demo video showing portfolio navigation.
  • Browser: Google Chrome / Firefox / Edge used to test the portfolio in real-time.
  • GitHub: To store project files online and share them with others.

These tools together helped in creating a *smooth, navigable, and professional digital portfolio* that can be viewed by students, teachers, and potential recruiters easily. Each technology was chosen to make the project *lightweight, responsive, and easy to maintain*.
