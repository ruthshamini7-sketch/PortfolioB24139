Project Model View

This project is designed to provide a clear and interactive digital portfolio. The main model of the project follows a *multi-page structure* where each page represents a specific section: Problem Statement, Project Overview, End User, Tools & Technologies, Design & Layout, Features & Functionality, Results & Screenshots, Conclusion, and GitHub Link.

The portfolio includes a navigation menu for seamless page transitions, images and videos to showcase results, and links to external resources like GitHub. The goal is to give viewers an *organized, professional, and visually appealing view* of the project while demonstrating interactivity and usability.
